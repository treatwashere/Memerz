# 🔊 Memerz (Beta)

Welcome to **Memerz**, a sleek, highly responsive web-based meme soundboard designed to deliver your favorite sound bites instantly. 

Currently in **Beta**, this project focuses on providing a clean, distraction-free user interface. It utilizes a custom application layout that mimics a native device screen, stripping away clunky window elements for an immersive, streamlined experience.

## ✨ Features
*   **Minimalist App Framing:** Built with a custom website device frame that skips standard browser clutter and address bars for a completely clean look.
*   **Instant Playback:** Zero-latency audio triggers powered by lightweight JavaScript.
*   **Responsive Layout:** Designed to scale smoothly across different screen dimensions.
*   **Pure Frontend:** Light, client-side architecture that runs fast without heavy backend overhead.

## 🛠️ Tech Stack
*   **HTML5:** Semantic structure for the soundboard layout and grid components.
*   **CSS3:** Styled with custom modern framing layouts for an elegant app aesthetic.
*   **JavaScript:** Optimized event handling for instant audio loading and playback execution.
